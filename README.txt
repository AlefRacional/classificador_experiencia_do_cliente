Classificador de experiencia do cliente usando Python

Hey Oh! Tudo ótimo. Neste projeto o objetivo é demonstrar como construir um classificador de sentimentos durante a experiencia do cliente no ato da compra ou no pós compra, utilizando Python e técnicas de Processamento de Linguagem Natural (NLP). 

Objetivo: Classificar frases como **positivas**, **negativas** ou **neutras** a partir de textos.

Etapas do Projeto

1. Coleta e limpeza de dados (dataset simulado)
2. Análise exploratória (visualização de dados)
3. Preparação de dados (vetorização)
4. Treinamento do modelo
5. Avaliação do modelo
6. Interface simples com Streamlit
7. Cuidados com segurança da informação
